This folder is where demo and user runs write their outputs by default.

Expected files from a successful run:

- `result_bundle.mat`
- `result_timeseries.csv`
- `summary_timeseries.png`
- `summary_spectra_and_closure.png`
- `debug_summary.mat`

This folder can also contain named subfolders for canned batches, for
example:

- `demo_run`
- `ow3d_compare_demo`
- `ow3d_kinematics_compare_demo`
- `bingchen_test_cases`

The Bingchen batch also writes:

- `bingchen_case_summary.csv`
- `bingchen_input_eta_overview.png`
- `bingchen_nonlinear_overview.png`
