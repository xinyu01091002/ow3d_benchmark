# Monochromatic Surface Horizontal Velocity Up To Third Order

This note derives the surface horizontal velocity for a monochromatic wave using the perturbation framework discussed in the MF12 paper.

In this note, the third-order first-harmonic correction to the potential is intentionally ignored. In other words, we do not include the `F_{13}` term from equation (3.67). We keep only:

- first-order potential: equation (3.3)
- second-order monochromatic transfer functions: equation (3.26)
- third-order monochromatic superharmonic transfer functions: equations (3.64) and (3.65)

For the final reported formulas, we also adopt a harmonic-truncation rule:

- at first order, keep only the first harmonic `\cos\theta`
- at second order, keep only the second harmonic `\cos 2\theta`
- at third order, keep only the third harmonic `\cos 3\theta`

So lower-harmonic contamination generated by the Taylor-expansion terms is shown in the derivation when useful, but it is not retained in the final truncated formulas.

The goal is to obtain the physical surface horizontal velocity

```math
\widetilde{u}(x,t)=\left.\frac{\partial \Phi}{\partial x}\right|_{z=\eta(x,t)}
```

for a single monochromatic component.

## 1. Setup

Consider a single monochromatic wave in one horizontal direction. Let

```math
\theta = \omega t-kx,
\qquad
k=\kappa_n,
\qquad
\omega=\omega_{1n},
\qquad
Z=z+h.
```

For simplicity, take the sine-cosine amplitudes to be

```math
a_n=a,
\qquad
b_n=0.
```

Then

```math
\eta^{(1)}=\varepsilon a\cos\theta.
```

The first-order potential from equation (3.3) is

```math
\Phi^{(1)}=\varepsilon F_1 \cosh(kZ)\,a\sin\theta,
```

with

```math
F_1=-\frac{\omega}{k\sinh(kh)}.
```

For the monochromatic self-interaction terms,

```math
A_{2}=\frac{a^2}{2h},
\qquad
A_{3}=\frac{a^3}{2h^2},
\qquad
B_2=B_3=0.
```

Hence

```math
\eta^{(2)}=\varepsilon^2 G_2 A_2 \cos 2\theta,
```

```math
\Phi^{(2)}=\varepsilon^2 F_2 \cosh(2kZ)\,A_2\sin 2\theta,
```

```math
\Phi^{(3)}=\varepsilon^3 F_3\cosh(3kZ)\,A_3\sin 3\theta.
```

The monochromatic second- and third-order transfer functions are

```math
G_2
=
\frac{1}{2}hk\left(2+\cosh 2kh\right)\frac{\coth kh}{\sinh^2 kh},
```

```math
F_2
=
-\frac{3}{4}\frac{h\omega}{\sinh^4 kh},
```

```math
F_3
=
\frac{1}{32}\frac{h^2 k\omega}{\sinh^7 kh}\left(-11+2\cosh 2kh\right).
```

## 2. Correct surface-velocity formula

The physical surface horizontal velocity is

```math
\widetilde{u}
=
\left.\Phi_x\right|_{z=\eta}.
```

Taylor-expanding about the mean water level `z=0` gives

```math
\widetilde{u}
=
\left.\Phi_x\right|_{z=0}
+
\eta\left.\Phi_{xz}\right|_{z=0}
+
\frac{1}{2}\eta^2\left.\Phi_{xzz}\right|_{z=0}
+
\cdots
```

Collecting by order in `\varepsilon`:

```math
\widetilde{u}^{(1)} = \left.\Phi_x^{(1)}\right|_{z=0},
```

```math
\widetilde{u}^{(2)} = \left.\Phi_x^{(2)}\right|_{z=0} + \eta^{(1)}\left.\Phi_{xz}^{(1)}\right|_{z=0},
```

```math
\widetilde{u}^{(3)}
=
\left.\Phi_x^{(3)}\right|_{z=0}
+
\eta^{(1)}\left.\Phi_{xz}^{(2)}\right|_{z=0}
+
\eta^{(2)}\left.\Phi_{xz}^{(1)}\right|_{z=0}
+
\frac{1}{2}\left(\eta^{(1)}\right)^2\left.\Phi_{xzz}^{(1)}\right|_{z=0}.
```

This is the proper perturbation-consistent route.

## 3. First-order surface horizontal velocity

Differentiate the first-order potential:

```math
\Phi_x^{(1)}
=
\varepsilon a F_1\cosh(kZ)\frac{\partial}{\partial x}\sin\theta
=
-\varepsilon a kF_1\cosh(kZ)\cos\theta.
```

Evaluating at `z=0`:

```math
\widetilde{u}^{(1)}
=
-\varepsilon a kF_1\cosh(kh)\cos\theta.
```

Substituting `F_1`:

```math
\widetilde{u}^{(1)}
=
\varepsilon a\omega\coth(kh)\cos\theta.
```

## 4. Second-order surface horizontal velocity

### 4.1 Direct second-order potential contribution

```math
\Phi_x^{(2)}
=
\varepsilon^2 A_2 F_2\cosh(2kZ)\frac{\partial}{\partial x}\sin 2\theta
=
-2k\varepsilon^2 A_2F_2\cosh(2kZ)\cos 2\theta.
```

At `z=0`:

```math
\left.\Phi_x^{(2)}\right|_{z=0}
=
-2k\varepsilon^2 A_2F_2\cosh(2kh)\cos 2\theta.
```

### 4.2 Surface-elevation correction to first order

```math
\Phi_{xz}^{(1)}
=
\frac{\partial}{\partial z}\Phi_x^{(1)}
=
-\varepsilon a k^2F_1\sinh(kZ)\cos\theta.
```

At `z=0`:

```math
\left.\Phi_{xz}^{(1)}\right|_{z=0}
=
-\varepsilon a k^2F_1\sinh(kh)\cos\theta
=
\varepsilon a k\omega\cos\theta.
```

Thus

```math
\eta^{(1)}\left.\Phi_{xz}^{(1)}\right|_{z=0}
=
\varepsilon^2 a^2 k\omega\cos^2\theta
=
\varepsilon^2 a^2 k\omega\frac{1+\cos 2\theta}{2}.
```

### 4.3 Second-order result

Therefore

```math
\widetilde{u}^{(2)}
=
-2k\varepsilon^2 A_2F_2\cosh(2kh)\cos 2\theta
+
\varepsilon^2 a^2 k\omega\frac{1+\cos 2\theta}{2}.
```

Using `A_2=a^2/(2h)`:

```math
\widetilde{u}^{(2)}
=
\varepsilon^2 a^2\frac{k\omega}{2}
+
\varepsilon^2 a^2
\left[
\frac{k\omega}{2}
-
\frac{kF_2\cosh(2kh)}{h}
\right]
\cos 2\theta.
```

Substituting `F_2`:

```math
\widetilde{u}^{(2)}
=
\varepsilon^2 a^2\frac{k\omega}{2}
+
\varepsilon^2 a^2 k\omega
\left[
\frac{1}{2}
+
\frac{3\cosh(2kh)}{4\sinh^4(kh)}
\right]
\cos 2\theta.
```

So the full retained second-order surface horizontal velocity contains:

- a mean contribution
- a second-harmonic contribution

Under the present harmonic-truncation rule, we keep only the `\cos 2\theta` part:

```math
\widetilde{u}^{(2)}_{\text{harmonic}}
=
\varepsilon^2 a^2 k\omega
\left[
\frac{1}{2}
+
\frac{3\cosh(2kh)}{4\sinh^4(kh)}
\right]
\cos 2\theta.
```

## 5. Third-order surface horizontal velocity

The third-order contribution is

```math
\widetilde{u}^{(3)}
=
\left.\Phi_x^{(3)}\right|_{z=0}
+
\eta^{(1)}\left.\Phi_{xz}^{(2)}\right|_{z=0}
+
\eta^{(2)}\left.\Phi_{xz}^{(1)}\right|_{z=0}
+
\frac{1}{2}\left(\eta^{(1)}\right)^2\left.\Phi_{xzz}^{(1)}\right|_{z=0}.
```

Each term is evaluated below.

### 5.1 Direct third-order potential contribution

From the third harmonic:

```math
\left.\Phi_{x,3}^{(3)}\right|_{z=0}
=
-3k\varepsilon^3 A_3F_3\cosh(3kh)\cos 3\theta.
```

Hence

```math
\left.\Phi_x^{(3)}\right|_{z=0}
=
-3k\varepsilon^3 A_3F_3\cosh(3kh)\cos 3\theta.
```

### 5.2 Mixed term `\eta^{(1)} \Phi_{xz}^{(2)}`

Differentiate the second-order potential:

```math
\Phi_{xz}^{(2)}
=
-4k^2\varepsilon^2 A_2F_2\sinh(2kZ)\cos 2\theta.
```

At `z=0`:

```math
\left.\Phi_{xz}^{(2)}\right|_{z=0}
=
-4k^2\varepsilon^2 A_2F_2\sinh(2kh)\cos 2\theta.
```

Multiply by `\eta^{(1)}=\varepsilon a\cos\theta` and use

```math
\cos\theta\cos 2\theta = \frac{1}{2}(\cos\theta+\cos 3\theta),
```

to obtain

```math
\eta^{(1)}\left.\Phi_{xz}^{(2)}\right|_{z=0}
=
-2\varepsilon^3 aA_2k^2F_2\sinh(2kh)\left(\cos\theta+\cos 3\theta\right).
```

Using `A_2=a^2/(2h)`:

```math
\eta^{(1)}\left.\Phi_{xz}^{(2)}\right|_{z=0}
=
-\varepsilon^3 a^3\frac{k^2F_2\sinh(2kh)}{h}\left(\cos\theta+\cos 3\theta\right).
```

### 5.3 Mixed term `\eta^{(2)} \Phi_{xz}^{(1)}`

Using

```math
\eta^{(2)}=\varepsilon^2 G_2A_2\cos 2\theta,
```

and

```math
\left.\Phi_{xz}^{(1)}\right|_{z=0}
=
\varepsilon a k\omega\cos\theta,
```

we get

```math
\eta^{(2)}\left.\Phi_{xz}^{(1)}\right|_{z=0}
=
\varepsilon^3 a k\omega G_2A_2\cos\theta\cos 2\theta
=
\varepsilon^3 a k\omega G_2A_2\frac{\cos\theta+\cos 3\theta}{2}.
```

So

```math
\eta^{(2)}\left.\Phi_{xz}^{(1)}\right|_{z=0}
=
\varepsilon^3 a^3\frac{k\omega G_2}{4h}\left(\cos\theta+\cos 3\theta\right).
```

### 5.4 Quadratic surface-elevation correction

Differentiate once more:

```math
\Phi_{xzz}^{(1)}
=
-\varepsilon a k^3F_1\cosh(kZ)\cos\theta.
```

At `z=0`:

```math
\left.\Phi_{xzz}^{(1)}\right|_{z=0}
=
\varepsilon a \omega k^2\coth(kh)\cos\theta.
```

Thus

```math
\frac{1}{2}\left(\eta^{(1)}\right)^2\left.\Phi_{xzz}^{(1)}\right|_{z=0}
=
\frac{1}{2}\varepsilon^3 a^3\omega k^2\coth(kh)\cos^3\theta.
```

Using

```math
\cos^3\theta = \frac{3\cos\theta+\cos 3\theta}{4},
```

we obtain

```math
\frac{1}{2}\left(\eta^{(1)}\right)^2\left.\Phi_{xzz}^{(1)}\right|_{z=0}
=
\varepsilon^3 a^3\omega k^2\coth(kh)\frac{3\cos\theta+\cos 3\theta}{8}.
```

## 6. Final third-order result

The third-order surface horizontal velocity can be written as

```math
\widetilde{u}^{(3)}
=
\varepsilon^3 a^3
\left[
\widehat{C}_1\cos\theta + \widehat{C}_3\cos 3\theta
\right],
```

where

```math
\widehat{C}_1
=
-\frac{k^2F_2\sinh(2kh)}{h}
+
\frac{k\omega G_2}{4h}
+
\frac{3}{8}\omega k^2\coth(kh),
```

and

```math
\widehat{C}_3
=
-\frac{3kF_3\cosh(3kh)}{2h^2}
-\frac{k^2F_2\sinh(2kh)}{h}
+
\frac{k\omega G_2}{4h}
+
\frac{1}{8}\omega k^2\coth(kh).
```

This reduced form shows the origin of each retained contribution:

- direct third-order superharmonic potential
- first-order surface acting on second-order potential
- second-order surface acting on first-order potential
- quadratic free-surface correction acting on first-order potential

Under the present harmonic-truncation rule, we drop the smaller third-order `\cos\theta` correction and keep only the `\cos 3\theta` part:

```math
\widetilde{u}^{(3)}_{\text{harmonic}}
=
\varepsilon^3 a^3 \widehat{C}_3 \cos 3\theta.
```

## 7. Why this differs from differentiating the surface potential

The MF12 paper also defines the surface velocity potential

```math
\widetilde{\Phi}(x,t)=\Phi(x,z=\eta,t),
```

with coefficients `\mu` in equations (3.77)–(3.81).

However,

```math
\frac{\partial \widetilde{\Phi}}{\partial x}
=
\left.\Phi_x\right|_{z=\eta}
+
\left.\Phi_z\right|_{z=\eta}\eta_x
=
\widetilde{u}+\widetilde{w}\eta_x.
```

Therefore

```math
\widetilde{u}
=
\widetilde{\Phi}_x-\widetilde{w}\eta_x.
```

So the derivative of the surface potential is not the physical horizontal surface velocity beyond first order.

At linear order they coincide, but from second order onward they differ by the geometric correction term `\widetilde{w}\eta_x`.

## 8. Summary

For a monochromatic wave, the correct surface horizontal velocity through third order is obtained by:

1. constructing `\Phi^{(1)}`, `\Phi^{(2)}`, `\Phi^{(3)}`
2. differentiating the bulk potential with respect to `x`
3. evaluating at the nonlinear surface by Taylor expansion about `z=0`
4. collecting terms up to `O(\varepsilon^3)`

The full asymptotic bookkeeping gives:

- `O(\varepsilon)` first harmonic
- `O(\varepsilon^2)` mean and second harmonic
- `O(\varepsilon^3)` first and third harmonics from the retained terms

But for the final working approximation in this note, we enforce the simpler harmonic-truncation rule:

- first order: keep only `\cos\theta`
- second order: keep only `\cos 2\theta`
- third order: keep only `\cos 3\theta`

Because the `F_{13}` term has been omitted on purpose, and because we are also intentionally discarding the smaller third-order `\cos\theta` correction generated by the Taylor-expansion terms, the final third-order formula below should be read as a truncated harmonic representation rather than the full third-order asymptotic result.

## 9. Explicit formulas summary

For quick reference, the truncated monochromatic surface horizontal velocity up to third order can be written explicitly as follows.

Let

```math
\theta = \omega t-kx.
```

Then

```math
\widetilde{u}
=
\widetilde{u}^{(1)}
+
\widetilde{u}^{(2)}
+
\widetilde{u}^{(3)}.
```

### First order

```math
\widetilde{u}^{(1)}
=
\varepsilon a \omega \coth(kh)\cos\theta.
```

### Second order

```math
\widetilde{u}^{(2)}
=
\varepsilon^2 a^2 k\omega
\left[
\frac{1}{2}
+
\frac{3\cosh(2kh)}{4\sinh^4(kh)}
\right]\cos 2\theta.
```

### Third order

With the third-order first-harmonic correction to the potential omitted, and with the additional harmonic-truncation rule that discards the smaller third-order `\cos\theta` correction, the retained third-order contribution is

```math
\widetilde{u}^{(3)}
=
\varepsilon^3 a^3
\widehat{C}_3\cos 3\theta,
```

```math
\widehat{C}_3
=
-\frac{3}{64}\omega k^2\frac{-11+2\cosh(2kh)}{\sinh^7(kh)}\cosh(3kh)
+
\frac{3}{2}k^2\omega \frac{\cosh(kh)}{\sinh^3(kh)}
+
\frac{1}{8}k^2\omega
\frac{(2+\cosh(2kh))\cosh(kh)}{\sinh^3(kh)}
+
\frac{1}{8}\omega k^2 \coth(kh).
```

### Combined expression

Putting the retained terms together:

```math
\widetilde{u}
=
\varepsilon a \omega \coth(kh)\cos\theta
+
\varepsilon^2 a^2 k\omega
\left[
\frac{1}{2}
+
\frac{3\cosh(2kh)}{4\sinh^4(kh)}
\right]\cos 2\theta
+
\varepsilon^3 a^3
\widehat{C}_3\cos 3\theta.
```

This is the truncated explicit expression corresponding to the present note, where:

- the third-order `F_{13}` correction has deliberately been excluded
- the second-order mean term has been dropped by choice
- the smaller third-order `\cos\theta` correction has also been dropped by choice
